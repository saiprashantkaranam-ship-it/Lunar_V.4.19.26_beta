# 🌙 Lunar AI — The Ultimate Local Brain

Lunar is a high-performance, private, and deeply reactive AI assistant designed to run entirely on your local hardware. No clouds, no tracking, just pure power.

## 🚀 Key Features

- **🧠 100% Local Intelligence**: Powered by Ollama and DeepSeek-R1. Your data never leaves your machine.
- **🎙️ Humanoid Synced Output**: Advanced text-to-speech synchronized with a word-by-word typing effect.
- **🛠️ Debug Console**: Integrated professional code editor for instant debugging and code analysis.
- **📷 Lunar Vision**: Instant camera activation with voice triggers and AI-driven image analysis.
- **😡 Angry Mode**: Toggle a sassier, more aggressive personality with a single click.
- **🎨 Dynamic Themes**: Choose from Neon, Sunset, Emerald, Ruby, and Gold aesthetics.
- **⚡ Zero-Delay Triggers**: Humanoid scripts for common questions (Who are you? Capabilities? etc.) work instantly without AI latency.
- **📦 Portable Build**: One-click scripts to package Lunar for use on any Windows machine.

## 🛠️ Tech Stack

- **Backend**: Node.js & Socket.io
- **AI Engine**: Ollama (DeepSeek-R1 1B/7B)
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), and Javascript
- **Bridge**: Python 3.x for system-level operations

## 🚀 Getting Started

### Prerequisites
1. [Node.js](https://nodejs.org/) (v18+)
2. [Ollama](https://ollama.com/)
3. [Python](https://www.python.org/)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/lunar-ai.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the setup script:
   ```bash
   ./setup.bat
   ```
4. Start the engine:
   ```bash
   npm run dev
   ```
